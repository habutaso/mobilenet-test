# mobilenet-test

# セットアップ
git, Pythonがインストールされていることを前提とする


### Windows
```powershell
> git clone https://github.com/habutaso/mobilenet-test
> cd mobilenet-test

# 仮想環境を作成
> python -m venv venv

# 仮想環境を起動
> . venv/Scripts/activate

# ライブラリをインストール
> pip install -r requirements.txt

# 実行
(venv) > python main.py


# 仮想環境から抜けるとき
(venv) > deactivate
> 
```
